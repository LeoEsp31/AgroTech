\# AgroTech - Sistema de Monitoreo de Riego Inteligente



Este proyecto surge como una solución para optimizar el uso del agua en la provincia de \*\*San Juan, Argentina\*\*, ante la crisis hídrica actual.



\##  Objetivo

Desarrollar un MVP para el monitoreo en tiempo real de la humedad del suelo en fincas locales utilizando sensores IoT y un backend robusto.



\##  Tecnologías

\* \*\*Lenguaje:\*\* Python

\* \*\*Framework:\*\* FastAPI

\* \*\*Arquitectura:\*\* Programación Orientada a Objetos (POO)

\* \*\*Control de Versiones:\*\* Git



\##  Contexto Académico

Proyecto desarrollado por un estudiante de 3er año de \*\*Ciencias de la Computación (UNSJ)\*\*.
